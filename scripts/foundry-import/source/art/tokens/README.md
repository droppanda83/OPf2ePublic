# Manual Token Art

Place creature token art files here to include them in the art pipeline.

## File naming convention
- Use the creature's normalized name: lowercase, spaces→hyphens
- Example: `goblin-warrior.webp`, `ancient-red-dragon.png`
- Supported formats: `.webp`, `.png`, `.jpg`, `.svg`

## Priority
Manual art takes highest priority — it overrides PF2e Foundry downloads.
Good for:
- Custom NPC art for your campaign
- Correcting/replacing auto-downloaded art
- Adding art for creatures not in the Foundry repo

## Sources to find free token art
- **Forgotten Adventures** (forgottenadventures.net) — Free tier has hundreds of tokens
- **2-Minute Tabletop** (2minutetabletop.com) — Free fantasy token packs
- **Token Stamp 2** (rolladvantage.com/tokenstamp) — Convert any image to a token
- **OpenGameArt.org** — CC-licensed fantasy art
- **Paizo Community Use** — Official PF2e art under community use policy
